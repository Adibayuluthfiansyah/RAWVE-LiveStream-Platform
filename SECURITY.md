---

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it privately to the maintainer to give us time to patch it before it becomes public knowledge.

**Contact:**
- Email: adibayu@adibayuluthfiansyah.dev
- GitHub: https://github.com/adibayuluthfiansyah

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if possible)

We will respond as quickly as possible and work collaboratively on a fix.
